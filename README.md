## Description
Your bridge from the model railroad to your microcontroller on a breadboard:

- Input: DCC signal
- Output: Railcom & DCC-ACK signals
- Power source (1): +5V 
- Power source (2): Recified railroad power

Built as breadboard friedly as possible.
